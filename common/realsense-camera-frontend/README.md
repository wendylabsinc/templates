# realsense-camera-frontend

React/Vite frontend for the `realsense-camera` Wendy template. This directory is the source of truth for the shared viewer UI. The same tree is vendored into `python/realsense-camera/`, `swift/realsense-camera/`, `rust/realsense-camera/`, and `cpp/realsense-camera/` because `wendy init` extracts only the selected language/template directory.

If you edit files here, re-copy the frontend files into each language template directory before shipping.
